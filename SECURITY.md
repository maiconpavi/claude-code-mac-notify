# Security Policy

## Supported Versions

Only the latest minor of `claude-code-mac-notify` receives security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you find a security issue, please **do not file a public GitHub issue**. Instead, email **maicon.pavi.vieira@gmail.com** with:

- A description of the issue.
- Steps to reproduce or a proof-of-concept.
- Affected version (commit SHA or release tag).
- Your assessment of the impact.

You can expect:

- An acknowledgement within 5 business days.
- A status update or fix plan within 30 days.
- Coordinated public disclosure once a fix is available.

Because this project ships shell scripts that run as part of your shell session, any code injection or escalation report is taken seriously.
